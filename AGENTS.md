# Vision
- To quiz students on their understanding of the book they choose, and keep a leaderboard of their scores 

# Homepage
- There should be a orange Login button at the top right corner of the screen and a orange Sign-Up button to the right of it. 
- Each time a user lands on the website a book reading quote should appear and you can choose a quote randomly from this URL: https://bookroo.com/blog/the-100-best-quotes-about-reading
- Creativly put ReadGenie as the title for the web page.
- Below the quote place a drop-down menu which has a list of book names. The book names will be stored in the data base. 

# Login
- When the user clicks the Login button they should be led to a screen which says enter username and password.
- Once they enter their username and password lead them back to the homscreen.
- There should be a button that says "Add book title" to the right of the dropdown menu.
- There should also be a forgot username and password button.

# Sign-up
- When the user clicks the Sign-Up button they should be led to a different page.
- There should sign up with their username as a vaild email, and a password of their choice.
- In the username box add an example that it has to be an email.

# Quiz
- There should be 10 multiple choice questions 
- Depending on the book, get questions from the Internet and the questions have to be random each time.
- Questions should only be about characters, plot, events, and timeline from the story.
- Do not repeat the book name in the questions.
- At the end of the quiz give a summary of which questions they got right and which qiestions they got wrong.
